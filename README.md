## 🚀 Запуск платформи

1. Склонуйте репозиторій:
   ```bash
   git clone https://github.com/username/repository.git
    ```
   
2. Перейдіть у папку проекту:
   ```bash
   cd repository
    ```
3. Встановіть залежності:
    ```bash
   pip install -r requirements.txt
   ```

4. Запустіть сервер:
    ```bash
   uvicorn main:app --reload
   ```
