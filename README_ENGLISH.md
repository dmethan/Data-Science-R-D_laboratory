## 🚀 Platform Launch

1. Clone the repository:
   ```bash
   git clone https://github.com/username/repository.git
    ```
   
2. Navigate to the project folder::
   ```bash
   cd repository
    ```
3. Install dependencies:
    ```bash
   pip install -r requirements.txt
   ```

4. Start the server:
    ```bash
   uvicorn main:app --reload
   ```
